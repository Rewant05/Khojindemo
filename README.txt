# 🔍 KHOJ – Centralized Lost & Found Platform 🇮🇳

KHOJ is a full-stack web platform that connects individuals with lost & found centers across India.  
It enables users to report lost or found items, track matches, and move towards recovery through a structured system.

---

## 🚀 Features

- 🧾 Report Lost Items  
- 📦 Report Found Items  
- 🔐 JWT-based Authentication  
- 📊 User Dashboard (track reports & matches)  
- 🔄 Basic Matching System (lost ↔ found)  
- 🛠 Admin Access (via role/email)  
- 📁 Image Upload Support  

---

## 🛠 Tech Stack

**Frontend**
- HTML, CSS, JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MySQL

**Authentication**
- JWT (JSON Web Tokens)
- Bcrypt (password hashing)
